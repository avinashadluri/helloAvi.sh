# AviWealth · Personal Finance Stack

Immigrant-friendly wealth building requires clarity. This trilogy of pages starts here.

## 1. Cashflow Command Center
- **Accounts**: Everyday + Bills + Savings + Investing, each tagged in Up/WeMoney.
- **Dashboard**: Google Sheet + Notion sync showing inflow/outflow, buffer, runway.
- **Automations**: Salary splits auto-transfer on payday; alerts if discretionary exceeds 30% envelope.

## 2. High-Yield Stack
- **High-interest**: ING, UBank, or Macquarie saver with bonus qualifications tracked via Shortcut.
- **ETFs**: DCA into VDHG + IVV (check tax residency). Use Pearler Autoinvest.
- **Super**: Compare fees, set after-tax contributions, ensure insurance cover appropriate.

## 3. Insurance Hygiene
- Life, TPD, Income Protection: review annually, note waiting periods.
- Health insurance: extras vs hospital; align with family needs.
- Document policies + claim instructions inside Notion vault accessible to partner.

## Resources
- Spreadsheet template + Loom walkthrough.
- n8n flow monitors FX rates + property listings for future purchases.
