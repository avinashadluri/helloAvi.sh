# FlowOS Workflows

Pick a lane below and remix the checklist to fit your context.

## 1. Builder Deep Work Loop
1. **Prime** (10 min) – journal, outline the problem, set SLO for the session.
2. **Sprint** (90 min) – single task, Pomodoro optional, queue Loom for demo.
3. **Debrief** (15 min) – note blockers, log learnings in Notion.
4. **Share** – Post snippet in Twitter threads or Linear for async feedback.

## 2. Ops for Parents
- **Sunday Summit**: 30-min family sync; review calendars, energy levels, logistics.
- **Flow Buckets**: label tasks as Energy, Admin, Creative, Rest.
- **Kid Collabs**: co-create Monday challenges (reading, STEM, art) in Notion.
- **Automations**: n8n sends morning briefing (school reminders + weather + lunch ideas).

## 3. Creator Content Engine
- **Capture**: Draft hooks in Craft/Notes while on walks.
- **Batch**: Record 3 reels / short videos every Thursday.
- **Edit**: Descript + CapCut templates.
- **Publish**: Buffer autopost + WhatsApp broadcast snippet.

## 4. FlowOS Retro
```md
- What gave energy?
- What drained energy?
- What can be automated?
- Who can help?
```
Run this weekly; feed actions into Linear board.
